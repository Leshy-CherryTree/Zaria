Editing Zone Libraries

What is a library?

Zone Library is a file that contains paths to Zone files containing game objects.

ApplicationState in Zaria Framework needs to get a library as input.

When creating a Game Map you will need to specify a library in your Work Files, so that the map knows where to get placable objects from.

Editing libraries

You can edit libraries only in text mode. It is possible to drag files from the file list into the text area.

Libraries in Work Files may contain files from the Assets folder. However libraries in Assets cannot contain files from Work Files.

It is valid to put paths to other libraries in a library. Be wary of circular references though.
